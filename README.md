## 使用方式

- 右键文件夹 执行  tinypng：fileImageCompression 进行文件内所有图片压缩（不支持文件夹递归压缩）
- 右键文件夹 执行  tinypng：fileImageDeepCompression 进行文件内所有图片压缩（压缩文件夹内所图片，包括文件夹内的所有文件夹）
- 右键文件 执行 tinypng：singleImageCompression 进行单个文件压缩
